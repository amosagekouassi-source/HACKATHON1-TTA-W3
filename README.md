▪︎PROJETC: Personalized AI-Powered Content Generator

Context: Hackathon Tech Talent Accelerator (TTA).

This project is a recommendation system that, thanks
to an AI, offers content to a user according to their
preference, activity history and activity hours.
It addresses a major problem that is information overload!
When you go on a platform and he has too much to choose
from at the same time, you can get lost and not want to continue.
Our system overcomes this melodrama because, by relying on the
activities of each user and the activity schedules according to
each profile, the system makes the right proposal at the right 
time! And avoids puzzles!Enabling a larger audience and more 

The code is in the HACKATHON folder, it is executable on Colab
and Vscode!
